# @raae/gatsby-plugin-starter

> Amazing plugin that solves everything

## How to install

`npm install @raae/gatsby-plugin-starter`

or

`yarn add @raae/gatsby-plugin-starter`

## How to use

```
module.exports = {
  plugins: [
    `@raae/gatsby-plugin-starter`
  ],
}
```
